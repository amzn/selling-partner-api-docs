レポートAPIとは？

レポートのSelling Partner API（レポートAPI）を使用して、出品パートナーが自らのビジネスを管理するのに役立つレポートを、Amazonから取得するアプリケーションを構築します。このAPIは、在庫の監視、フルフィルメントの注文の追跡、税金情報の取得、返品と出品者のパフォーマンスの追跡、フルフィルメント by Amazonでの出品ビジネスの管理など、さまざまなユースケースのレポートを提供します。レポートAPIのオペレーションと関連データタイプ、スキーマについて詳しくは、レポートAPIリファレンスを参照してください。使用可能なレポートタイプについては、reportTypeの値を参照してください。

レポートを生成するための3つの主要なワークフローは、レポートのリクエスト、レポートのスケジュール設定、自動生成されたレポートの取得です。

レポートのリクエスト

createReportオペレーションを使用することで、使用可能なレポートタイプをオンデマンドでリクエストできます。次を参照してください：チュートリアル： レポートのリクエスト（手順について）。

レポートのスケジュール設定

createReportScheduleオペレーションを使用して、スケジュールに従ってAmazonからレポートリクエストが自動で送信されるようにできます。次を参照してください：チュートリアル： レポートのスケジュール設定と取得（手順について）。

自動生成されたレポートの取得

Amazonでは、いくつかのレポートが自動的に生成されます。次を参照してください：チュートリアル： 自動生成されたレポートの取得（自動生成されたレポートを取得するための手順について）。
